# Guia de revision editorial

## Criterios

1. Coherencia canonica del universo.
2. Claridad narrativa por pagina.
3. Coherencia entre texto y prompt.
4. Progresion visual consistente entre paginas.

## Regla original/current

- `original`: conservar propuesta base para comparativa.
- `current`: version adaptada aprobada para publicacion.

## Imagenes

- `main` representa la lectura principal de la pagina.
- `secondary` solo si agrega valor (detalle, atmosfera o contexto).
- Alternativas se guardan como candidatas; una queda activa por slot.
- Prompts finales deben converger a plantilla estructurada v1:
  - `SUJETO`
  - `ESCENA`
  - `ESTILO`
  - `COMPOSICION`
  - `ILUMINACION_COLOR`
  - `CONTINUIDAD`
  - `RESTRICCIONES`

## Resultado esperado

- Cuento navegable en webapp sin perdida de trazabilidad editorial.
