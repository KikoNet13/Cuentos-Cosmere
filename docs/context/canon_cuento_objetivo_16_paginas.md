# Canon del cuento objetivo (perfil recomendado de 16 páginas)

## Fuente visual de referencia

- Carpeta base: `docs/assets/style_refs/Hansel y Gretel/`
- El ejemplo se toma como referencia de composición y estilo.
- La página de actividades no forma parte del producto objetivo.

## Alcance del canon

1. Este documento define una estructura editorial recomendada.
2. No impone un límite técnico fijo de páginas.
3. La cantidad real depende del archivo canónico del cuento (`NNN.md`).

## Estructura editorial sugerida

1. Portada.
2. Página de título.
3. Páginas narrativas mixtas.
4. Páginas de imagen completa en momentos clave.
5. Contraportada común para la saga.

## Criterios visuales

1. Estilo de ilustración infantil consistente.
2. Paleta y textura estables en todo el cuento.
3. Identidad visual fija de personajes clave.
4. Longitud y densidad textual adecuadas a lectura infantil.
